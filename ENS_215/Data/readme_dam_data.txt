# Variables in Dams dataset
# Original data source: http://www.fao.org/nr/water/aquastat/dams/index.stm

### Variable list ###
# Country: Country dam is located in           
# Dam_name: Name of the dam         
# ISO_code: Country name abbreviation         
# River: River flowing into the reservoir
# Major_basin: River basin that dam and reservoir are located in  
# Year_completed: Year dam construction was completed  
# Dam_height_m: Dam height in meters 
# Reservoir_capacity_million_m3: Storage capacity of the reservoir in units of millions of cubic meters 
# Reservoir_area_km2: Surface area of the reservoir in square kilometers  
# Latitude: Dam location longitude coordinates
# Longitude: Dam location longitude coordinates       
# Continent: Continent dam is located in             
# Sedimentation_percent: Percent of reservoir that has been filled in with sediment    
# Irrigation: Dam/reservoir used for irrigation (1 = yes, 0 = no)      
# Water_supply: Dam/reservoir used for water supply (1 = yes, 0 = no)    
# Flood_control: Dam/reservoir used for flood control (1 = yes, 0 = no)       
# Hydroelectricity: Dam/reservoir used for hydroelectricty generation (1 = yes, 0 = no)    
# Navigation: Dam/reservoir used to control water levels in river to allow for navigation in the river (1 = yes, 0 = no)          		  
# Recreation: Dam/reservoir used for recreation (1 = yes, 0 = no)               
# Pollution_control: Dam/reservoir used for pollution control (1 = yes, 0 = no)             
# Livestock_rearing: Dam/reservoir used for livestock rearing (1 = yes, 0 = no)             
# Other: Dam/reservoir used for other purposes (1 = yes, 0 = no)
